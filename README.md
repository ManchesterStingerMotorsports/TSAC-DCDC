# TSAC-DCDC
Low power HV-LV flyback converter to power the TSAC's internal safety circuitry.

Input voltage:
	Min: 35V to allow for single segment operation
	Max: 600V to allow for use across accumulator types

Estimated Output @ 12V with minimal ripple:
	6W continuous
	24W Peak
